# LoRA Mask Loss Exporter

Photoshop CEP 面板插件 — 一键将选区导出为 LoRA 掩码损失训练数据集。

## 训练时的损失计算

```
loss_final = loss_raw × mask
```

| mask 值 | 效果 |
|---------|------|
| 255（白/不透明） | 正常训练该像素 |
| 0（黑/透明） | 忽略该像素 |

## 两种掩码模式

| | Alpha 掩码 (A通道) | Conditioning 掩码 (C通道/R通道) |
|---|---|---|
| **使用的通道** | A = Alpha（透明度） | R = Red（红色通道值） |
| **选区内的像素** | 保留原图内容（不透明） | RGB(255,255,255) 纯白 |
| **选区外的像素** | 全透明（alpha=0） | RGB(0,0,0) 纯黑 |
| **边缘处理** | 羽化边缘 → 半透明过渡 | 羽化边缘 → 灰度过渡 |
| **输出** | 带透明度的 PNG | 三通道 RGB PNG (R=G=B) |

## 导出目录结构

每次导出同时产生三个子目录：

```
{输出目录}\
  ├── source\    ← 等比缩放后的完整原图（不含选区）
  ├── alpha\     ← Alpha 掩码
  └── mask\      ← Conditioning 掩码
```

三个目录中输出文件均为**源文件名.png**，确保训练时 source 与 mask 尺寸严格一致。

## 安装

1. 下载或 Clone 本仓库
2. 双击运行 `install.ps1`（推荐）或 `install.bat`
3. 安装脚本会**自动识别** Photoshop 安装路径（注册表 → Program Files → AppData 回退）
4. 重启 Photoshop
5. 菜单：窗口 → 扩展 → LoRA Mask Loss

## 使用流程

1. **选择目录** — 浏览本地图片库，显示目录树和缩略图
2. **双击缩略图** — 在 PS 中打开图片
3. **框选关注区域** — 在 PS 中做选区（支持羽化）
4. **选择模式** — Alpha 掩码 / Conditioning 掩码
5. **设置参数** — 原始尺寸 / 百分比 / 固定最长边(256~4096)
6. **点击导出** — 一键导出到 source/ alpha/ mask/ 三个目录

## 功能

- 📁 目录树浏览，支持懒加载子文件夹
- 🖼️ 缩略图网格，60~200px 可调滑块，自动均分列宽
- 🔍 默认显示所有文件，可按格式筛选（PNG/JPG/PSD/TIFF/WebP/RAW 等）
- 📊 四种排序方式（名称/修改时间/大小/类型）
- 🏷️ 导出状态徽章（✔已导出 / ⚠失败）
- 📝 运行日志面板
- 💾 UI 参数自动记忆（输出路径、缩放、模式等）
- 📐 固定最长边模式：超过设定值才压缩，不超过保持原尺寸（不放大）
- 🔧 自动识别 PS 安装目录，无需手动配置路径
- 🚫 不依赖任何第三方库，纯 Adobe CEP + Node.js 内置模块

## 要求

- Adobe Photoshop 2020+（CSXS 9.0）
- Windows 系统
- CEP Debug Mode 已启用（安装脚本自动设置）
